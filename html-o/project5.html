My First Unity & C#

Game

Made in

June 2020

5. My First Game: This project is something I always wanted to try. By using a course on Udemy, I was finally able to create a game. I used Photoshop and Unity with C#. The interesting part of this whole project is the fact that all graphics, which includes the characters and background of the game, were made by using a mouse and keyboard. Thanks to the feature 'smoothing' in Photoshop, I was able to create the main character, enemies, and the background of the game by using a mouse and not a drawing tablet. I do think a drawing tablet would have been way faster and more versatile.
